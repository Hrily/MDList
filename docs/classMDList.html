Detailed Description

template<class T>
class MDList< T >

MDList class MDList is a dictionary based datastruture which stores (key, value) pairs.

It has a root node. Opoerations performed on a MDList are insert, find, remove. MDList is Lock-Free datastructure, so it is thread safe.
